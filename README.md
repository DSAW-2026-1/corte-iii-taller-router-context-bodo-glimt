# React Router App

This is a simple React application implementing a router with Login, Home, and About pages.

## Pages

- **Login**: Initial page with a login button that navigates to Home.
- **Home**: Displays a link to the About page.
- **About**: Information page.

## How to Run

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:5173`.

## Navigation Flow

- Start at Login page.
- Click "Login" to go to Home.
- From Home, click "Go to About" to visit the About page.
